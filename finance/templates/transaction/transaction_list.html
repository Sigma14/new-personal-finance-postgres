Account Transaction Description
This feature allows you to seamlessly sync your account with your bank using Plaid's secure API. With Plaid, you can automatically download your transaction information while ensuring your financial data is protected. Learn more about Plaid's security and functionality here: What is Plaid?
Upload button Description
The 'Upload' option enables users to upload a bulk list of transactions using our provided format. This is particularly useful for those who prefer a manual approach or have a large number of transactions to record at once, streamlining the process while maintaining accuracy.
Add_transaction_Account Description
The '+ Add Transaction' feature allows for manual entry of individual transactions. This is ideal for users who prefer a hands-on approach or need to enter transactions not captured through automatic sync or bulk upload. The process is straightforward: simply click the button, fill in the transaction details like amount, date, category, and any relevant tags or notes, and save. This feature ensures even the smallest transactions are accounted for in your financial tracking.
Transaction Description
  • The 'Transaction List Table' is a comprehensive display of all your financial transactions. It's where you can view, sort, and analyze your spending and income. Key features of this table include:
  • Date Range Selection: Select specific date ranges to focus on certain periods. This tool is invaluable for tracking your spending habits over time or reviewing transactions for a particular month or week.
  • Report Button: Once you've chosen a date range, use the 'Report' button to refresh the table with transactions from that period. This ensures you're always analyzing the most current and relevant data.
  • Filter by Tags: This functionality allows you to filter transactions based on the tags you've assigned. Whether you're looking for all transactions tagged as 'Groceries' or 'Utilities,' this feature makes it simple to find exactly what you're looking for.
Transaction Graph Description
The 'Transaction Graph' displays two bar graphs for each day, representing the total amount of debit and credit transactions. This visual differentiation between incomes (credit) and expenditures (debit) provides an immediate understanding of your daily financial flow, making it easier to identify spending trends and manage your finances more effectively.
